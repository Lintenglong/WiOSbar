# FluidBar Plugin Suggestions

除了已有的剪贴板、媒体、Agent 状态、Windows 通知四个内置插件，以下是一些值得考虑的扩展方向：

## 高优先级（推荐优先实现）

### 1. 系统性能监视 (System Monitor)
显示 CPU 使用率、内存占用、GPU 温度等实时数据。展开卡片可显示详细曲线图。
- **数据源**: `System.Management`（已有引用）或 `PerformanceCounter`
- **触发场景**: 每 2-3 秒轮询，常驻可选
- **适用性**: 开发者、重度用户常用

### 2. 天气显示 (Weather)
在灵动岛上显示当前天气、温度、空气质量。展开卡片可看未来几小时预报。
- **数据源**: 免费天气 API（如 OpenWeatherMap、和风天气）
- **触发场景**: 每 15-30 分钟刷新
- **适用性**: 所有用户日常使用

### 3. 番茄钟计时 (Pomodoro Timer)
内置番茄工作法计时器，25 分钟专注 + 5 分钟休息循环。展开卡显示剩余时间和当前阶段。
- **数据源**: 本地计时，无需网络
- **触发场景**: 用户手动启动，定时推送状态
- **适用性**: 高效工作流

## 中优先级

### 4. Wi-Fi / 蓝牙设备详情
当前网络 SSID、信号强度、IP 地址。蓝牙连接设备名称和电量。
- **数据源**: Windows Network API, `Windows.Devices.Bluetooth`
- **触发场景**: 网络/蓝牙状态变化时
- **适用性**: 网络管理场景

### 5. 屏幕截图 / 录屏状态
截图后显示缩略图和快捷操作（复制/编辑/标注），录屏时显示计时。
- **数据源**: 监听剪贴板图片或截图快捷键
- **触发场景**: 截图/录屏事件触发
- **适用性**: 内容创作者

### 6. 日程提醒 (Calendar)
同步 Windows 日历或 Outlook，显示即将到来的日程和提醒。
- **数据源**: `Windows.ApplicationModel.Appointments`
- **触发场景**: 日程开始前 5-15 分钟提醒
- **适用性**: 办公用户

## 低优先级（创意方向）

### 7. 股票/加密货币行情
实时显示自选股票或加密货币价格涨跌。
- **数据源**: 免费行情 API
- **触发场景**: 定时刷新（注意 API 频率限制）
- **适用性**: 投资者

### 8. GitHub 通知
显示 GitHub issues、PR 审查请求、Star 等消息。
- **数据源**: GitHub API / Octokit SDK
- **触发场景**: 轮询 + Webhook
- **适用性**: 开源贡献者

### 9. 快递追踪
绑定快递单号，在灵动岛上显示物流更新。
- **数据源**: 快递 100 等聚合 API
- **触发场景**: 物流更新时推送
- **适用性**: 日常购物用户

### 10. 单词学习卡片
定时推送一个单词/短语，帮助碎片化学习。
- **数据源**: 本地词库 JSON
- **触发场景**: 每 30-60 分钟推送一次
- **适用性**: 学习者

## 插件开发原则

1. **不阻塞 UI 线程**: 使用 `DispatcherTimer` 或 `Task.Run` 异步采集
2. **优雅降级**: API/权限不可用时静默降级，不影响其他插件
3. **少即是多**: 每个插件专注一个场景，不过度复杂
4. **尊重隐私**: 网络请求类插件不收集个人信息
5. **低功耗**: 轮询间隔不低于 2 秒（除媒体这类需要亚秒级响应的）
