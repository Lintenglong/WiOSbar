## 🎨 功能亮点展示

### 亮点 1：四源歌词自动选择
```
用户播放音乐
    ↓
检测播放器（Spotify/酷狗/网易云/QQ音乐）
    ↓
提取歌曲名 + 艺术家
    ↓
优先级策略：
  Kugou API → 成功？→ 显示歌词
       ↓ 失败
  网易云 API → 成功？→ 显示歌词
       ↓ 失败
  QQ音乐 API → 成功？→ 显示歌词
       ↓ 失败
  Spotify API → 成功？→ 显示歌词
       ↓ 失败
  显示「纯音乐，请欣赏」
```

### 亮点 2：iOS 式智能聚合
```
音量快速调节：
  第1次：显示「音量 50%」
  第2次：（800ms内）→ 缓存
  第3次：（800ms内）→ 缓存
  第4次：（800ms内）→ 缓存
  第5次：（800ms内）→ 聚合
  结果：显示「音量调节 (x5)」
```

### 亮点 3：专注模式自动检测
```
前台窗口变化
    ↓
检测条件：
  ├─ 窗口尺寸 ≈ 屏幕尺寸？→ 全屏应用
  ├─ 进程名包含 game/steam？→ 游戏进程
  ├─ 浏览器全屏？→ 视频播放
  └─ 专注助手启用？→ Windows 专注模式
    ↓
任一条件满足 → 隐藏灵动岛
    ↓
所有条件不满足 → 恢复显示
```

### 亮点 4：数据驱动优化
```
使用统计收集
    ↓
生成洞察报告：
  ├─ 最活跃来源：volume (34%)
  ├─ 日均事件：21 次
  ├─ 媒体播放时长：45 小时
  └─ 构建成功率：87%
    ↓
用户洞察：
  "本周你触发了 342 次音量调节"
  "最常用的音乐应用是 Spotify"
```
